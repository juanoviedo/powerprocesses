Plasma CNC Cutter

Cali, 2022

It was sought to manufacture a high-speed machine to improve the cutting of sheet thicknesses less than 2 mm. It also sought to eliminate downtime in the cutting process to increase productivity. Peak Hour Alarm

Cali, 2021

Is a device designed to be installed in the car. It uses the ESP32 WiFi microcontroller to notify the user of the peak hour. This device notifies the users before they start their cars in the peak hour time frame. This way, users can avoid penalty fees. The project needed a lot of investment to go into serial production, so the project is paused for now. Also, there are other solutions, like mobile apps.
